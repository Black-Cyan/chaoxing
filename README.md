# 超星学习通AI自动答题 

> 市面上不少 AI 答题脚本不仅要付费，提示词质量还参差不齐，加上我本身就有可用的模型，索性自己写了个支持自定义模型的自动答题插件，彻底摆脱收费困扰

## 快速开始

- 在 [脚本猫](https://scriptcat.org) 安装该脚本
- 打开学习通答题页面
- 配置 `baseUrl` `apiKey` `模型`
- 开始使用吧！
